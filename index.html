XML-lib is a simple xml file to object convertor for c++ application, XML-lib use not any DOM for interprate XML file.

XML-lib is split in 3 part :

  • XMLRoot for manipulate XML data
  • XMLBase for save and load data from XML file
  • XMLUtility for search in data

On this web site you find all documentation and example about the library.
